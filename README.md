# Student Management System

A Python/Flask web app with SQLite database.

## Setup & Run

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Run the app
python app.py

# 3. Open browser
http://127.0.0.1:5000
```

## Login
- Username: `admin`
- Password: `admin123`

## Features
- Admin login/logout
- Add, edit, delete students
- Search students
- Mark daily attendance (Present/Absent)
- Attendance report with percentage
- Dashboard with stats
